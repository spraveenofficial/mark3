I will try to explain this by providing easy to understand examples and simplified concepts. Let's jump to the main topic of this article.

What is Axios?

Axios, is a javascript library used to make HTTP requests (Restful or GraphQL API) from node.js or XMLHttpRequest from the browser. Axios accomplies the same task as the fetch API provide by the many browser.

Prerequisites:

In order to use Axios you first need to have Node.js installed in your local machine / computer. In case if you didn't have installed yet please click on this link. This link will take you to the official Node.js website.

Next, we must install axios package in project. To get this done, just run the following command in your terminal.

npm install axios

In case, if you use yarn, then follow this command:

yarn add axios

Using Axios

The main reason is to use Axios instead of fetch API is because it provides a very clean and precise syntax. Nevertheless, it’s important to acknowledge that Axios is not always an ideal solution, and there are tons of better options are there for making HTTP requests.

Code using Axios

Yeah, now we are are good to go with Axios. We are going to write our first line of code below. Follow the code below:

Firstly we need to import the Axios package in our project (index.js).

const axios = require('axios');

Now, we have imported the Axios package successfully. We are going to hit an dummy api to get your all confusing rid out of your mind.

const getRequest = () =>{
axios.get('https://jsonplaceholder.com')
.then(response => console.log(response.data))
.catch(error => console.log(error));
}

getRequest();

In the above example, I am making GET request to the dummy api to obtain a object / list. To use Axios library we call the axios function and then we call any of the methods provided by the Axios library which represent the different HTTP verbs. You need to learn about HTTP requests type. Since Axios works with Promises the code on line 3 in the given above example will return a Promise.
